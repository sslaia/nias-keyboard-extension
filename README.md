## nias-keyboard-extension

This extension creates a new "Input Method" for ChromeOS. You will need to install it and then activate it in your settings.

## Install the Unpacked Extension

1. Download and save the zip file somewhere.

2. Open a new tab in Chrome and go to chrome://extensions.

3. In the top-right corner of the extensions page, toggle on "Developer mode".

4. A new set of buttons will appear. Click the "Packed extension" button.

5. In the file dialog, select the zip file you saved before and click "Open".

6. If successful, you will see a new "Nias keyboard extension" card on your extensions page.

## Enable the new Input Method

1. Click the clock in the bottom-right corner, then click the "Settings" (gear) icon.

2. In the Settings app, go to Advanced > Languages and inputs.

3. Click on Input method > Manage input methods.

4. Scroll through the list until you find "Nias keyboard extension". Click the checkbox next to it to enable it.

## How to Use

1. Switch Input Methods when you want to type in Nias. Click on the little language code in the bottom-right corner of your screen (it probably says "US").

2. In the pop-up list, select "Nias keyboard extension".

3. Type! You can now type in any text field.

* Press Right Alt + o to get ö.

* Press Right Alt + w to get ŵ.

* All other keys will work as normal.

4. To switch back to your regular keyboard, just click the language code again and select your original layout. (You can also use Ctrl + Space or Ctrl + Shift + Space to quickly cycle through your enabled keyboards).
